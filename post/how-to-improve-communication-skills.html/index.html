    沟通能力太烂，面对陌生人话都不敢多说两句。要是碰到自己喜欢的女孩，你说面对面的时候怎么良好进行自我介绍，这时候估计脑子大概率又是一片空白。为了提高自己的沟通能力，奶爸老师刚好出了一套沟通课，可能偏实践，可能单纯看书效果会好很多，不多说，赶紧学起来。下面的知识点来自奶爸的课程。

    沟通到底是一门怎样的艺术？

    首先需要明确的是，社交是一门体力活。单纯的沟通技巧，强大的心理因素都不是阻碍提高沟通的注意因素，自身的体力才是。举个生活例子，假如你在一个众多陌生人的社交环境中，此时你进行沟通是需要耗费大量精力的，如果身体跟不上，再多的沟通技巧，强大的心理抗压能力，也会阻碍良好沟通的进行。

    那该如何提高自己的体力？

    1，高能量姿势

    • 叉腰昂首站立姿势

    • 反复深呼吸

    • 适度运动

    2，充足的睡眠习惯

    自我介绍应该怎么做？

    不知道有没有发现平时沟通的时候存在一个细节，别人看着我们的时候我们并不会感觉到有压力，但是轮到我们去注视对方的时候压力却特别大。但是说话时候注视对方这是一个沟通底线，别人说话可以不进行眼神的注视，但轮到自己说话时必须需要抬起头注视大家。

    那该如何进行自我介绍？

    首先，自我介绍是区分场合的，主要是两大块，正式的&非正式的场合。

    正式场合可以运用在一般的商务，演讲等，主要表示内容为 姓名 + 工作 + 成就性事件(代表性事件)，介绍事件时候多使用数字，更具有说服性。正式场合的自我介绍建议一年更新一次，背到滚瓜烂熟，好在沟通场合使用下意识来进行自我介绍。

    那么相对的非正式场合的自我介绍，姓名+工作+兴趣爱好，通过兴趣爱好来继续宁
